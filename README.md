# Querying with SQL & Python

This project is part of the [Backend Roadmap](https://roadmap.sh/projects/sql-and-python). It involves connecting Python to a SQLite database (Chinook) to perform data analysis and answer business-related questions.

This code demonstrates my skills in:

* **Database Connectivity:** Interfacing Python with SQLite databases using the native `sqlite3` library.
* **SQL Proficiency:** Writing complex queries using `JOIN`, `GROUP BY`, and `ORDER BY` to extract meaningful insights.
* **Data Integration:** Moving and transforming data from SQL databases into Pandas DataFrames for analysis.
* **Data Visualization:** Translating raw query results into visual charts using Matplotlib.

## Project Scope

* **Database:** Utilized the Chinook sample database to model a digital music store.
* **Exploration:** Mapped database schemas and relationships to optimize query performance.
* **Business Intelligence:** Answered key business questions regarding top-selling tracks, revenue by country, and employee sales performance.

## Technologies Used

* Python
* SQLite3
* Pandas
* Matplotlib
* Jupyter Notebook
